# Color System

Use a calm, professional palette.

## Recommended base palette
- Background: `#F7F8FA`
- Surface: `#FFFFFF`
- Surface muted: `#F1F3F6`
- Border: `#E2E8F0`
- Text primary: `#0F172A`
- Text secondary: `#475569`

## Primary accents
- Primary navy: `#1E3A5F`
- Primary muted: `#2F4F6F`
- Accent blue-gray: `#5B7A99`

## Semantic colors
- Success: `#2E7D5A`
- Warning: `#B7791F`
- Danger: `#B04A4A`
- Info: `#3B6EA8`

## Rules
- Do not oversaturate.
- Do not use more than one primary accent family.
- Keep badges readable and calm.
- Do not use gradients for operational surfaces.
