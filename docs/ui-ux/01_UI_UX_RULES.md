# UI UX Rules

## Core principles
- Desktop first
- Table first
- Simple hierarchy
- Minimal visual noise
- Clear statuses
- Fast scanning
- Fast updating

## Primary pages
- Dashboard
- Clients
- Intakes
- Projects
- Tasks
- Deliverables
- Team
- Settings

## Shared patterns
- consistent page headers
- search + filters near top
- action buttons on the right side of headers
- reusable status badges
- empty states with next-step guidance
- right-side drawers for quick create and quick edits where practical
