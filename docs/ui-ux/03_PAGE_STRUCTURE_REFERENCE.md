# Page Structure Reference

## Standard page layout
1. Page title and subtitle
2. Main actions on the right
3. Search and filters row
4. Main data area
5. Empty state if no data
6. Pagination or secondary controls if needed

## Project detail layout
- Header summary
- quick actions
- tabs
- overview
- tasks
- deliverables
- files
- team
- activity

## Dashboard layout
- KPI cards
- needs attention section
- urgent work lists
- recent activity
- workload summary
