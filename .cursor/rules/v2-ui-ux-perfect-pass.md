Sprint 06 UI/UX rule set for ReKa Engineering OS V2.

Use only this core visual palette:
- Carbon Black: #1D1F1E
- Porcelain: #FFFDF7
- Dark Wine: #851E1E
- Twilight Indigo: #142D50
- Graphite: #454040
- Parchment: #F4F3EE

Color rules:
- Twilight Indigo is the primary action and navigation accent
- Dark Wine is only for urgent, revision, critical, and destructive states
- Carbon Black and Graphite are for text, structure, and dark surfaces
- Porcelain and Parchment are for backgrounds and cards
- Keep the interface mostly neutral
- Accent colors must be used sparingly

Typography:
- Primary font: Inter
- Strong readable hierarchy
- Avoid tiny text

Layout rules:
- Desktop-first
- Table-first
- Strong visual hierarchy
- Improve spacing, readability, and consistency
- Preserve current architecture
- No full redesign

Forbidden:
- no gradients
- no glassmorphism
- no flashy startup visuals
- no rainbow charts
- no decorative widgets
- no unrelated feature additions
- no Google Drive
- no notifications
- no analytics expansion
- no new modules
